# AllyClicker — чек-лист ручного тестирования (Фаза 5.3)

Проходи сверху вниз, отмечай `[x]`. Всё — курсором/головой, без физических кнопок.
Если что-то не так — пометь ❌ и опиши, что произошло.

## 1. Панель и базовый dwell
- [ ] Панель появляется, тёмная тема, кнопки видны
- [ ] Наведение на кнопку → красная плашка «заезжает» под неё (арм) + звук арма
- [ ] LEFT: навёл на LEFT → задержался на цели на экране → одиночный клик
- [ ] RIGHT: контекстное меню появляется
- [ ] DOUBLE: двойной клик (например, выделение слова / открытие)
- [ ] Звук клика при срабатывании
- [ ] Мазок по панели мгновенно снимает активную функцию (плашка гаснет)
- [ ] Припаркованный курсор НЕ «строчит» кликами (одно срабатывание на остановку)

## 2. DRAG
- [ ] Арм DRAG → задержка на старте (нажатие) → перемещение → задержка на конце (отпускание)
- [ ] Выделение текста работает
- [ ] Выделение области (рамкой) на рабочем столе работает

## 3. MIDDLE (Scroll & Links)
- [ ] Над ссылкой в браузере → открывается в новой вкладке
- [ ] Над пустым местом страницы → старт авто-скролла (появляется/понятен якорь)
- [ ] Увёл курсор от якоря → скролл в ту сторону; дальше = быстрее
- [ ] Остановил курсор → срабатывает левый клик и скролл прекращается
- [ ] Направление скролла естественное (туда, куда ведёшь)

## 4. Перемещение и сворачивание панели
- [ ] Арм DRAG → задержка на ON/OFF → панель следует за курсором → остановка = отпустил
- [ ] Позиция запоминается после перезапуска приложения
- [ ] Клик по ON/OFF (без DRAG) → панель сворачивается до одной кнопки; ещё раз → разворачивается
- [ ] В горизонтальной ориентации сворачивание/разворачивание тоже работает

## 5. Idle-disarm
- [ ] Заармить действие, не двигать курсор ~2 мин → функция снимается сама (плашка гаснет)

## 6. Настройки — вкладка Behavior
- [ ] Меняю AutoMouse Delay → Save → скорость срабатывания меняется вживую
- [ ] Panel button delay, Drag press/release — применяются
- [ ] Default to Left Click / Automatic Cancel — поведение меняется
- [ ] Idle-disarm (минуты) — применяется
- [ ] Jitter tolerance / Move threshold — влияют на устойчивость к тремору
- [ ] Auto-scroll Intensity — скорость скролла меняется
- [ ] Sound feedback выкл → звуки пропадают
- [ ] Launch at login — появляется/исчезает в «Объектах входа»

## 7. Настройки — вкладка Panel
- [ ] Orientation Vertical/Horizontal → панель перестраивается; горизонтальная встаёт сверху по центру
- [ ] Тумблер кнопки off → кнопка исчезает с панели (на своём месте в списке остаётся)
- [ ] Тумблер обратно on → кнопка возвращается на прежнюю позицию
- [ ] Стрелки ↑/↓ меняют порядок активных кнопок
- [ ] ON/OFF всегда первая; её стрелки заблокированы; выключить можно
- [ ] Убрал ON/OFF → секция «Launch collapsed» гаснет и сбрасывается
- [ ] Icon style Custom/System — иконки в списке и на панели меняются
- [ ] Icon size — глиф масштабируется
- [ ] Panel width / Opacity — применяются вживую
- [ ] Launch collapsed = on → полный перезапуск → панель стартует свёрнутой

## 8. Настройки — общее
- [ ] Save применяет и закрывает окно
- [ ] Cancel закрывает без применения
- [ ] Reset to defaults возвращает значения (позиция панели не прыгает)
- [ ] Окно настроек открывается на прежнем месте после перемещения+закрытия
- [ ] Вкладка About: иконка, версия, кликабельные ссылки; футер (Reset/Cancel/Save) скрыт

## 9. Система
- [ ] Иконка приложения видна в Finder / «Программах»
- [ ] Значок в меню-баре (курсор), меню: Settings… / Quit
- [ ] Quit из меню-бара закрывает приложение (синтетические кнопки не залипают)
- [ ] После перезагрузки Мака (если Launch at login вкл) — приложение стартует само

## 10. Установка (опционально)
- [ ] `brew install --cask allyclicker` ставит рабочую копию (проверено ранее)
