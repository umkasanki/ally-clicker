# PNC macOS — иконка строки меню (status item)

Отдельная иконка для верхней строки меню (menu bar extra / `NSStatusItem`) — указатель-стрелка, силуэтом, как у DwellClick. Поставляется **отдельно** от набора кнопок панели кликов: у строки меню свои правила.

## Что в комплекте

- `cursor.svg` — исходный глиф, `viewBox 0 0 18 18`, монохром на `currentColor`.
- `MenuBarIcon.xcassets/cursor.imageset` — векторный PDF + `Contents.json` (Preserve Vector Data + Render As: Template Image).
- `preview.png` — как иконка выглядит на светлой и тёмной строке меню при @1x (18px) и @2x (36px).

## Канон строки меню (чем отличается от кнопок панели)

- **Template-изображение (силуэт).** Иконки строки меню не цветные: система сама красит силуэт чёрным на светлой теме и белым на тёмной, плюс подсветка при открытом меню. Поэтому глиф — сплошная заливка на прозрачном фоне, никаких контуров и цветов. Флаг `template-rendering-intent: template` это и задаёт.
- **Свой размер.** Рисуется под высоту строки меню — артворк ~15 pt в холсте 18×18 pt (с небольшими полями). Это меньше и другой пропорции, чем кнопочные 56-pt глифы, поэтому это отдельный файл, а не переиспользование панельного.
- **Вектор → Retina.** PDF хранит вектор; система растеризует под @2x/@3x на лету, так что на Retina силуэт всегда чёткий.

## Как подключить (NSStatusItem)

```swift
let statusItem = NSStatusBar.system.statusItem(withLength: NSStatusItem.variableLength)
if let button = statusItem.button {
    let img = NSImage(named: "cursor")   // имя imageset
    img?.isTemplate = true               // уже задано в каталоге; дублировать не вредно
    button.image = img
}
```

В template-режиме система сама управляет цветом и подсветкой — вручную ничего красить не нужно. Если хотите задать точный размер в строке меню, можно выставить `img?.size = NSSize(width: 18, height: 18)` (или 16, под вкус), вектор останется резким.

## Заметки

- Силуэт проверен на светлой и тёмной строке меню при @1x/@2x (см. `preview.png`) — читается как указатель.
- Если позже захотите анимировать индикатор (например, моргание при активном dwell), это делается сменой `button.image` или `contentTintColor`, но базовый ассет к этому уже готов.
